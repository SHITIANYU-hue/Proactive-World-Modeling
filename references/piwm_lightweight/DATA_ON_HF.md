# PIWM Lightweight Data

This directory imports the lightweight PIWM reference from:

- Source repository: https://github.com/guochenmeinian/piwm
- Source branch: main
- Source commit: 2f356c8a00a5d434b6c8c5bbe4a979277ff1ebfc

The full data payload is stored in the Hugging Face dataset repository:

- Dataset: https://huggingface.co/datasets/GameFreshMan/PIWM
- Imported prefix: https://huggingface.co/datasets/GameFreshMan/PIWM/tree/main/piwm_lightweight
- Video assets: https://huggingface.co/datasets/GameFreshMan/PIWM/tree/main/piwm_lightweight/data/videos

GitHub keeps the reference docs, scripts, labels, manifests, prompts, and seed/eval metadata. Large video files are intentionally not tracked here.
