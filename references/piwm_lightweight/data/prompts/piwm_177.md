# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色休闲裤，背一个黑色双肩包，一只手拿着手机，另一只手扶着背包肩带
人物背景：刚放学路过校园便利区的男大学生，经过智能售货设备前时被短暂吸引但没有真正停下。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：attention
- Belief：他意识到面前可能是一个可以快速购买饮料或零食的设备，但还没有形成具体兴趣或选择判断。
- Desire：他主要想继续赶路回宿舍，只是顺便确认这里是否有值得以后停下来的东西。
- Intention：他倾向于不操作、不询问、不停留，保持经过状态并继续向前走。

# Observable Performance
- 主要可见行为：他从画面一侧缓慢经过正面区域，眼神短暂扫向设备方向，步伐略微放慢但没有停住，也没有伸手触碰或做出扫码、点击动作。
- 面部状态：表情自然平静，眉眼稍有注意力集中，带有短暂好奇但没有明显兴奋或犹豫。
- 身体姿态：身体保持行走中的轻微前倾，肩膀放松，正面朝向镜头经过，双手都在画面内，一手握手机，一手轻扶背包肩带。

# 10-Second Timeline
- 0–2 秒：20岁左右的男大学生从画面一侧进入，正面朝向镜头缓步经过，手里拿着手机，另一只手扶着黑色双肩包肩带。
- 2–5 秒：他脚步略微放慢，眼神从前方移向镜头下方的设备方向，像是刚注意到这里有什么可购买的东西。
- 5–8 秒：他保持安静经过，没有停下，也没有伸手操作，只是用平静的表情快速打量了一下。
- 8–10 秒：他收回视线，步伐恢复正常，继续从画面另一侧离开，整个过程没有与设备发生互动。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
