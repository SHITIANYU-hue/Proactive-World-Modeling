# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
24岁左右的女性，齐肩黑色微卷发，穿浅米色针织开衫和浅蓝色牛仔裤，斜挎一个小号棕色帆布包，右手拿着手机。
人物背景：周末在商场便利区闲逛、顺路看看零食饮品的年轻女性。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前可选的商品可能适合周末休闲时随手购买，整体选择过程清楚顺畅。
- Desire：她想轻松浏览并找到一个自己感兴趣、适合当下心情的小零食或饮品。
- Intention：她倾向继续安静浏览几秒，确认偏好的选项后再决定是否购买。

# Observable Performance
- 主要可见行为：她正面站在镜头前，视线平稳地在不同高度和左右方向之间移动，偶尔轻轻点头，双手始终在画面内，右手自然握着手机，左手轻扶斜挎包带，浏览动作连续且没有停顿。
- 面部状态：表情放松自然，眼神专注但不紧张，嘴角带轻微笑意，没有困惑或犹豫过重的神情。
- 身体姿态：身体姿态直立放松，肩膀自然下沉，上半身略微前倾，重心稳定，整体呈现轻松感兴趣的浏览状态。

# 10-Second Timeline
- 0–2 秒：这位24岁左右的女性正面站定，右手拿着手机，左手扶着棕色帆布包带，视线自然抬起开始浏览前方。
- 2–5 秒：她的眼睛平稳地从左到右扫过，嘴角保持轻微笑意，上半身略微前倾，像是在顺畅比较几个感兴趣的选择。
- 5–8 秒：她轻轻点了一下头，视线移到稍低的位置，右手手机保持在胸前附近，左手仍自然扶着包带，没有表现出困惑或停顿。
- 8–10 秒：她重新看向正前方某个位置，表情保持轻松专注，身体稳定站立，像是准备继续安静浏览并确认偏好的选项。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
