# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
26岁左右的女性，黑色中长发低马尾，穿浅灰色西装外套和白色衬衫，搭配黑色直筒裤，左肩背一个米色通勤包，手里拿着手机。
人物背景：下班路过办公楼便利区的年轻女白领，想买点零食缓解饥饿但还在比较选择。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前某款零食看起来比较适合下班后垫肚子，但还不确定是否比其他选择更合适。
- Desire：她想快速找到一款不太负担、口味合适、能缓解饥饿的零食。
- Intention：她倾向继续观察并等待一个温和推荐来帮助自己形成初步购买倾向。

# Observable Performance
- 主要可见行为：她多次将视线投向同一位置，又短暂移开比较其他方向，右手拇指轻轻摩挲手机边缘，偶尔抬手停在胸前像是在准备选择但没有真正动作。
- 面部状态：表情专注而略带犹豫，眉头轻微收紧，嘴唇自然闭合，眼神在感兴趣和不确定之间来回变化。
- 身体姿态：身体正面朝向镜头，肩膀略向前倾，双脚站定，重心偶尔轻微左右转移，双手始终在画面内。

# 10-Second Timeline
- 0–2 秒：这位26岁左右、扎低马尾的女白领走近后停下，正面看向镜头下方的选择区域，左肩的米色通勤包自然垂在身侧。
- 2–5 秒：她的视线反复落在同一处，随后又移向旁边位置作比较，右手握着手机，拇指轻轻摩挲手机边缘。
- 5–8 秒：她微微前倾，眉头轻收，抬起右手停在胸前像要做决定，但又放慢动作，继续看回那款让她感兴趣的零食方向。
- 8–10 秒：她保持正面站姿，轻轻抿唇并点头幅度很小，仍未明确下手，表现出需要温和推荐来推动初步倾向的状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
