# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
58岁左右的女性，短卷发，穿米色针织开衫和深绿色长裤，肩背一个棕色小皮包，手里拎着一个浅色购物袋
人物背景：周末逛街途中路过便利区的中老年女性，被智能零售设备吸引而停下观察。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：attention
- Belief：她认为眼前的智能设备可能可以买东西，但暂时不清楚应该从哪里开始看或怎么操作。
- Desire：她想先弄明白这个设备能提供什么，并找到一个简单明确的关注点。
- Intention：她倾向于继续停留几秒观察，等待或寻找一个简单提示来帮助自己决定先看哪一类商品。

# Observable Performance
- 主要可见行为：她停在镜头前，视线在正前方小范围左右移动，偶尔低头看自己的手和购物袋，又重新抬头观察，动作放慢并带有中等程度犹豫。
- 面部状态：眉头轻微皱起，眼神有些茫然，嘴唇微抿，表情克制但明显困惑。
- 身体姿态：身体正面朝向镜头，上半身略微前倾，双手都在画面内，一只手握着购物袋提手，另一只手轻轻扶着包带，站姿稳定但不完全放松。

# 10-Second Timeline
- 0–2 秒：58岁左右的女性从画面前方停下，正面看向镜头方向，短卷发和米色针织开衫清晰可见，手里的浅色购物袋轻轻晃了一下。
- 2–5 秒：她微微前倾，视线在正前方左右扫动，眉头轻皱，像是在确认这里到底能看什么或该从哪里开始。
- 5–8 秒：她低头看了一眼手中的购物袋，又抬头望向前方，另一只手轻轻捏住棕色小包的肩带，表情仍然有些茫然。
- 8–10 秒：她保持站定，嘴唇微抿，眼神停在正前方某一点，像是在等待一个简单提示来帮她明确关注方向。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
