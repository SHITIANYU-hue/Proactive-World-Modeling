# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
20岁左右的女性，黑色中长发自然披肩，穿浅灰色连帽卫衣和蓝色牛仔裤，肩背一个米白色帆布包，手里拿着手机和校园卡
人物背景：下课后路过校园便利区的女大学生，想买点东西补充能量但还没决定选哪一种。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为面前有好几个选择都挺吸引人，但各自可能在口味、价格和饱腹感上有所不同。
- Desire：她想在课后快速选到一个既合口味又不太贵、还能缓解饥饿感的选项。
- Intention：她倾向继续在几个选项之间对比，并需要先确认自己最在意的是口味、价格还是饱腹感再做决定。

# Observable Performance
- 主要可见行为：她的视线在左右几个方向之间来回扫视，偶尔低头看手机又抬头比较，手指轻轻摩挲校园卡，动作有停顿但不慌张。
- 面部状态：表情专注中带着中等程度的犹豫，眉头轻微收紧，嘴唇自然抿住，眼神在比较时略显摇摆。
- 身体姿态：身体正面朝向镜头，肩膀略微前倾，双手保持在胸前到腰部范围内，一只手拿手机和校园卡，重心在双脚之间轻微转换。

# 10-Second Timeline
- 0–2 秒：这名20岁左右、穿浅灰色连帽卫衣的女大学生正面站定，抬眼看向前方，手机和校园卡都拿在画面内。
- 2–5 秒：她的视线从左侧移到右侧又回到中间，像是在被多个选择吸引，眉头轻轻皱起，手指无意识摩挲校园卡边缘。
- 5–8 秒：她微微前倾，低头快速看了一眼手机后再次抬头，眼神在几个方向之间来回停留，比较的意味更明显。
- 8–10 秒：她抿了抿嘴，重心轻轻换到另一只脚上，仍正面看着前方，没有立刻行动，像是在等待确认自己最该优先考虑什么。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
