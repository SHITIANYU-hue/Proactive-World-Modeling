# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
28岁左右的男性，短黑发，穿浅蓝色衬衫和深灰色休闲西裤，肩上背一个黑色通勤双肩包，手里拿着一部手机但未举起遮挡面部。
人物背景：第一次来到这处便利零售点、下班途中想买点东西的年轻白领男性。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：attention
- Belief：他意识到这里有多种可选商品类别，但暂时不清楚应该先从饮料、零食还是简餐方向开始看。
- Desire：他想快速找到一个符合当前需求的购买方向，避免在设备前花太久犹豫。
- Intention：他倾向于先停下来观察整体选项，并希望得到一个开放式引导来帮助自己聚焦需求。

# Observable Performance
- 主要可见行为：他在设备前站定后没有立即操作，目光在不同区域之间来回扫视，手指轻轻摩挲手机边缘，偶尔小幅抬手又放下，表现出中等程度的犹豫。
- 面部状态：表情克制而略显困惑，眉头轻微收紧，嘴唇自然闭合，眼神持续搜索但不焦急。
- 身体姿态：身体正面朝向镜头，双脚站稳，肩膀略向前倾，双手保持在胸前到腰部之间的画面范围内，整体姿态停顿而谨慎。

# 10-Second Timeline
- 0–2 秒：这名28岁左右的男性走到画面中央并停下，正面朝向镜头，双肩包背在肩上，手机自然握在一只手里。
- 2–5 秒：他微微抬头，目光从左到右缓慢扫过前方区域，眉头轻轻收紧，像是在确认这里有哪些可选方向。
- 5–8 秒：他短暂低头看了一眼手里的手机，又把视线移回前方，手指摩挲手机边缘，另一只手小幅抬起后停住没有继续动作。
- 8–10 秒：他保持站定，身体略向前倾，视线再次在前方来回移动，表情显示他还没有决定从哪个商品类别开始。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
