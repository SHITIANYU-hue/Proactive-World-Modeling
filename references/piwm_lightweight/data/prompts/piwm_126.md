# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
28岁左右的男性，短黑发，穿浅蓝色衬衫和深灰色西裤，胸前挂工牌，左手拿着手机，右肩背一个黑色通勤包
人物背景：午饭时间短暂经过便利区的年轻男职员，原本只是路过但在智能售货设备前放慢脚步。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：attention
- Belief：他意识到自己正经过一个可快速购买午餐或饮品的零售点，但还没有真正查看设备内容。
- Desire：他想在有限的午休时间里快速判断这里是否值得停下来。
- Intention：如果收到清晰友好的欢迎提示，他倾向于把目光从前方通道转向设备并开始关注。

# Observable Performance
- 主要可见行为：他从路过状态减速并在镜头前短暂停住，双手都在画面内，左手握着手机，视线起初仍越过设备看向前方通道，随后才轻微回落到镜头方向。
- 面部状态：表情中性略带匆忙，眉眼放松但注意力尚未集中，末尾出现轻微被吸引的停顿感。
- 身体姿态：身体正面朝向镜头但重心略向一侧，肩膀带着行走后的惯性，右手自然扶着背包肩带，姿态保持克制和犹豫。

# 10-Second Timeline
- 0–2 秒：28岁左右的男性男职员从正面进入画面并放慢脚步，左手拿着手机，目光仍看向前方通道。
- 2–5 秒：他在设备前短暂停住，身体正对镜头，右手扶着黑色通勤包肩带，但眼神还没有落到设备内容上。
- 5–8 秒：他像是接收到一句简短欢迎提示，脚步完全停下，眉头轻微放松，视线开始从远处通道向镜头方向回收。
- 8–10 秒：他微微抬起下巴并稳定站姿，目光终于落到设备前方，呈现出刚被吸引注意、准备进一步查看的状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
