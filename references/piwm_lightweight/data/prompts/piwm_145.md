# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
26岁左右的女性，齐肩黑色直发，穿浅灰色西装外套和白色衬衫，搭配深色直筒裤，左手拎一杯外带咖啡，右肩背一个米色通勤托特包
人物背景：拎着咖啡路过便利零售区、被智能设备外观吸引而短暂停下的年轻女白领

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：attention
- Belief：她觉得眼前的零售设备外观看起来新颖，可能有方便的东西，但还不清楚具体值得看什么。
- Desire：她想在不耽误太多时间的情况下，快速判断这里是否有和自己当前需求相关的选择。
- Intention：她倾向于先停留几秒随意观察，并需要一个定向提问来帮助自己聚焦需求。

# Observable Performance
- 主要可见行为：她在镜头前自然停下，左手稳稳拿着咖啡，目光在不同方向来回扫视，偶尔轻微抬眉和偏头，但没有持续盯住某一个固定区域。
- 面部状态：表情平静，带一点好奇，嘴唇自然闭合，犹豫感较低，没有明显焦虑或强烈购买冲动。
- 身体姿态：身体正面朝向镜头，站姿放松，肩膀自然下沉，重心略微前移，头部随视线小幅左右转动，双手始终在画面内。

# 10-Second Timeline
- 0–2 秒：这位26岁左右的女性白领拎着咖啡从画面前方路过并放慢脚步，目光被前方吸引后停在镜头前。
- 2–5 秒：她正面站定，左手拿着咖啡，眼睛从左侧扫到右侧，又轻轻抬头看向上方，像是在确认这里大致有什么。
- 5–8 秒：她微微偏头，眉毛轻抬，视线继续在几个方向之间移动，没有在任何位置停留太久。
- 8–10 秒：她保持放松站姿，咖啡杯停在胸前，脸上仍是好奇但未决定的表情，像是在等待一个问题来帮她明确想看什么。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
