# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
45 岁左右的女性，短发微卷，穿浅灰色针织开衫和米白色衬衫，手腕戴银色手表，左手拿着手机；旁边站着一位年龄相近的女同事，齐肩直发，穿深绿色衬衫，手里拿着帆布包带子。
人物背景：午休后和同事一起路过办公楼便利区的中年女性，正在智能零售设备前比较饮品选择但意见不一致。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前可选项都可能合适，但自己和同事关注点不同，暂时难以判断哪一个更符合当下需求。
- Desire：她想在口味、提神效果和是否适合下午继续工作之间找到一个共同优先级。
- Intention：她倾向先听取设备提出的聚焦问题，再和同事缩小选择范围。

# Observable Performance
- 主要可见行为：她和同事都正面看向前方，视线在不同位置之间来回移动，偶尔互相侧眼确认；她用手机轻轻点着掌心，又抬手做小幅度比较手势，几次想伸手但又停住。
- 面部状态：眉头轻微皱起，嘴唇抿住后又短暂张开，表情专注但犹豫，同事也带着不确定的思考神情。
- 身体姿态：她身体微微前倾，双肩略收，双手保持在胸前可见范围内；同事站在她右侧半步，身体同样朝前但头部偶尔偏向她。

# 10-Second Timeline
- 0–2 秒：45 岁左右的短发女性和右侧女同事一起停在镜头前，两人正面看向前方，她的手机握在左手里，表情开始变得专注。
- 2–5 秒：她的目光在前方不同位置之间来回移动，右手抬起做了一个小幅度比较动作，同事轻轻摇头并把手放在帆布包带子上。
- 5–8 秒：她侧眼看向同事，嘴唇无声动了几下，随后又转回正面，眉头更明显地皱起，伸出的右手停在半空没有继续。
- 8–10 秒：两人仍未形成一致意见，她把右手收回胸前，和同事一起安静看向前方，像是在等待设备用问题帮助她们聚焦需求。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
