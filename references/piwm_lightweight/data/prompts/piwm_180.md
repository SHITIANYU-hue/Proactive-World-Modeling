# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
26岁左右的女性，黑色中长发低马尾，穿浅灰色针织开衫和白色衬衫，搭配深色直筒裤，肩上背一个米色帆布托特包，双手自然放在身前
人物背景：午休时间顺路来便利零售区闲逛的年轻女职员，已经在某款商品前停留了一会儿，正在安静确认是否符合自己的需求。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前这款商品可能适合午休后使用，但还需要再确认关键信息是否匹配自己的偏好。
- Desire：她想在不被打扰的情况下快速核对信息，判断这款商品是否值得继续考虑。
- Intention：她倾向于继续安静停留片刻，保持观察和确认，暂时不需要进一步操作或外部介入。

# Observable Performance
- 主要可见行为：她始终正面站在镜头前，视线稳定地在前方略低处小范围移动，偶尔轻微点头，双手保持在画面内并安静握在身前，没有明显求助或操作动作
- 面部状态：表情专注而平静，眉眼微微收敛，嘴唇自然闭合，犹豫感很低，呈现认真确认信息的状态
- 身体姿态：身体站姿稳定，肩膀放松，上半身略微前倾，重心均匀，整体姿态克制安静，保持持续停留

# 10-Second Timeline
- 0–2 秒：26岁左右的女性正面站定在镜头前，双手自然握在身前，视线落在前方略低处，表情专注安静。
- 2–5 秒：她的目光缓慢左右移动，像是在逐项确认信息，肩膀放松，身体保持原位不动。
- 5–8 秒：她轻轻点了一下头，嘴唇仍然闭合，眼神稳定，表现出已经基本理解但仍在确认细节的状态。
- 8–10 秒：她继续安静停留，双手保持在画面内，姿态平稳，没有伸手操作或寻求帮助，只维持专注观察。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
