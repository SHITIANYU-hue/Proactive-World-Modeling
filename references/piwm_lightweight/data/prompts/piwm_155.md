# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
25岁左右的女性，黑色中长发，穿米色针织开衫和浅灰色长裤，肩上背一个棕色托特包，双手自然垂放。
人物背景：在商场便利区短暂停留、正在比较饮品选择但有些拿不定主意的年轻女性

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前有几个选择都可能合适，但不确定哪一个能同时满足自己想要健康和女儿想要口味的需求。
- Desire：她希望尽快找到一个母女都能接受、不会引发争执的选择。
- Intention：她倾向于继续比较并等待设备通过询问偏好来帮她们聚焦共同需求。

# Observable Performance
- 主要可见行为：母亲和女儿都正面站在镜头前，视线在不同方向之间来回移动，母亲偶尔抬手轻点下巴又放下，女儿微微摇头并用手指向另一侧，两人短暂停顿后互相看一眼。
- 面部状态：母亲眉头轻皱、嘴唇微抿，表现出明显犹豫；女儿眼神认真但略带坚持，表情有些拿不定主意。
- 身体姿态：母亲身体略向前倾，肩膀放松但停顿感明显，一只手扶着包带；女儿站在她右侧稍靠前，身体也朝向镜头，双手始终在画面内。

# 10-Second Timeline
- 0–2 秒：42岁左右的女性带着约走近并停在镜头前，两人同时抬眼打量前方，母亲一手扶着棕色托特包带。
- 2–5 秒：母亲视线在左右之间来回移动，轻轻抿嘴并用手指碰了碰下巴，女孩则看向另一侧并小幅度摇头。
- 5–8 秒：女孩抬手指向自己更感兴趣的方向，母亲看了她一眼后又转回前方，眉头略皱，显得仍在比较。
- 8–10 秒：母女两人并排停住，双手都留在画面内，母亲轻轻点头又迟疑地停下，像是在等待设备进一步询问她们共同关注的需求。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
