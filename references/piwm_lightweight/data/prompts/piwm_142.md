# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
两人均为 25 岁左右，男性短黑发，穿浅灰色连帽卫衣和深色休闲裤，手里拿着手机；女性长发披肩，穿米白色针织开衫和浅蓝牛仔裤，单肩背一个小号棕色包。
人物背景：周末逛街途中路过便利区的一对年轻情侣，临时停下来看是否需要买点东西。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：attention
- Belief：他们意识到这里可能能满足临时补给需求，但还不确定自己更需要解渴、提神还是简单垫肚子。
- Desire：他们想先弄清彼此当前最在意的需求方向，再决定是否继续选择。
- Intention：他们倾向继续观察并等待对方先表达偏好，如果被提问引导会更容易聚焦需求。

# Observable Performance
- 主要可见行为：两人并肩停在镜头前，视线在前方不同区域来回扫视，偶尔互相侧眼看一眼，手部动作克制，手机和包带都保持在画面内，没有伸手操作的明确动作。
- 面部状态：两人表情自然偏中性，带轻微好奇和中等程度犹豫，嘴角放松，眉头偶尔轻微收紧。
- 身体姿态：两人身体正面朝向镜头，肩膀略向前，站姿放松但有轻微重心转换，彼此距离较近，像是在等对方先开口。

# 10-Second Timeline
- 0–2 秒：这对年轻情侣从画面前方停下，男性拿着手机垂在胸前，女性扶着包带，两人同时抬眼看向前方。
- 2–5 秒：男性的视线从上方扫到下方后短暂停住，女性也跟着打量前方，两人都没有立刻伸手操作。
- 5–8 秒：女性轻轻偏头看向男性，男性也回看她一眼，双方表情都有些犹豫，像是在等对方先决定想要什么。
- 8–10 秒：两人重新面向前方，男性轻微抿嘴，女性手指捏住包带，继续观察但仍未表现出明确选择倾向。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
