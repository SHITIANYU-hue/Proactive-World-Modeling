# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
25岁左右的女性，黑色中长发低马尾，穿浅灰色通勤西装外套和白色衬衫，深色直筒裤，肩上背一个黑色通勤包，手里拿着手机和小型门禁卡套。
人物背景：下班赶回家途中路过办公楼便利区的年轻女职员，已经基本决定买一件常买的补给品。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：她认为自己已经找到了合适的选择，只需要再确认一次是否符合下班路上快速购买的需求。
- Desire：她想尽快完成最后确认并买到这件补给品，然后继续赶回家。
- Intention：她倾向于保持当前选择，短暂停顿后直接进入购买确认动作。

# Observable Performance
- 主要可见行为：她正面站在镜头前，视线稳定地在正前方小范围移动，双手始终在画面内，手机握在一只手中，另一只手短暂靠近胸前后放松，整体是目的性强的 Hold:ambient 式短暂停留，没有招手、求助或四处寻找工作人员的动作。
- 面部状态：表情平静专注，眉头轻微收拢，嘴唇自然闭合，犹豫感很低，像是在做最后确认。
- 身体姿态：身体微微前倾但站姿稳定，肩膀略带下班后的疲惫感，重心固定在双脚之间，动作克制而明确。

# 10-Second Timeline
- 0–2 秒：25岁左右的女性正面走入并停在镜头前，黑色通勤包挂在肩上，手中握着手机，目光迅速落向正前方。
- 2–5 秒：她保持稳定站姿，视线在正前方小范围扫过，眉头轻轻收拢，像是在核对已选内容是否正确。
- 5–8 秒：她微微前倾，拿手机的手保持在胸前高度，另一只手自然悬停后放松，表情专注且没有明显犹豫。
- 8–10 秒：她轻轻点一下头，视线固定，身体仍正面朝向镜头，呈现出准备继续完成购买确认的状态。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
