# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
28岁左右的男性，短黑发，穿浅蓝色衬衫和深灰色西裤，胸前挂着工牌，左手拿着手机，右手自然垂在身前。
人物背景：午休时顺路在办公楼便利区购买午饮的年轻男性职员，已完成扫码支付并在设备前等待出货。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：action
- Belief：他认为扫码支付已经成功，饮品很快就会取到，这次购买流程顺利可靠。
- Desire：他想尽快拿到午休饮品，然后回到工位继续休息或工作。
- Intention：他倾向于在原地轻松等待出货，并用点头微笑表示感谢后结束本轮交互。

# Observable Performance
- 主要可见行为：他看向镜头方向下方的设备交互区域，短暂确认手机后放低手臂，随后保持等待状态，并在最后轻轻点头示意感谢。
- 面部状态：表情放松满意，嘴角微微上扬，眼神稳定，没有明显犹豫或焦虑。
- 身体姿态：身体正面朝向镜头，站姿自然端正，肩膀放松，双手始终在画面内，重心稳定地停留在设备前。

# 10-Second Timeline
- 0–2 秒：这名28岁左右的男性职员正面站在镜头前，低头看了一眼左手手机，像是在确认支付完成。
- 2–5 秒：他把手机稍微放低，视线回到镜头方向下方的设备区域，脸上露出轻松满意的微笑。
- 5–8 秒：他保持端正站姿安静等待，右手自然垂在身前，左手握着手机，几乎没有犹豫动作。
- 8–10 秒：他轻轻点头并微笑示意感谢，身体仍正面朝向镜头，表现出这轮购买交互已经顺利完成。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 允许出现与当前状态一致的轻微手机支付、刷卡、确认、等待出货或取货动作；动作应克制、真实，不要夸张贴屏、反复敲击或展示界面细节
- 不展示机器主动响应或屏幕 UI 内容；即使发生支付、刷卡、等待出货或取货，也只表现顾客自然动作，不展示二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
